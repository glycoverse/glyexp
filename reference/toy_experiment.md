# Toy experiment

A toy experiment object for new users to play with to get familiar with
[`experiment()`](https://glycoverse.github.io/glyexp/reference/experiment.md)
objects.

## Usage

``` r
toy_experiment
```

## Format

### `toy_experiment`

An
[`experiment()`](https://glycoverse.github.io/glyexp/reference/experiment.md)
object with 4 variables and 6 samples. `var_info` contains fields:
`protein`, `peptide`, and `glycan_composition`. `sample_info` contains
fields: `group` and `batch`.
